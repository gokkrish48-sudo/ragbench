System Topology

1. Retrieval Tier

Multi-vector HNSW indexing with metadata-aware filtering. Uses bi-encoders for sub-50ms latency at scale.

2. Reasoning & Eval

Cross-encoder reranking followed by LLM-as-a-judge faithfulness scoring via asynchronous evaluation workers.
